FAT 16 filesystem reader provides API for reading fat16 img files including parsing file alocation table, traversing file clusters reading directories, file contents and metadata
with simple cli for testing
cli commands:
cd DIRECTORY_NAME - change current directory to chosen folder
ls - list current directory files
preview FILENAME - preview first 100 bytes of file
